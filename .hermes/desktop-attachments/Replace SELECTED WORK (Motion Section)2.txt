Plan: Replace SELECTED WORK (Motion Section) with Claude Design
Summary
Replace the current components/sections/Motion.jsx — a light-paper sticky section with a flying WebGL carousel + SMIL SVG fallback — with the Claude Design's dark-themed horizontal card rail. No flying animation; pure CSS scroll-snap carousel with hover lift effects.

What changes
1. Rewrite components/sections/Motion.jsx (~350 → ~120 lines)
Remove entirely:

All WebGL/SVG flying carousel logic (FlightCard, ProjectCard, createFlyingCarouselLayout, createMotionStudyTiming, THREE camera projection, SMIL timeline seeking, motionFlight singleton integration, useCarouselMount, ticker/gsap subscriptions)
The sticky +=400% pinned ScrollTrigger (the section becomes a normal non-sticky section)
Build instead:

A static <section id="motion" data-nav-tone="dark"> with dark background (#04060c + subtle blue radial gradient)
A header row: left side = eyebrow ("Named client record") + h2 ("Real names. Real businesses. No invented case studies.") + right side = "View all work →" link to /work
A horizontal flex card rail: overflow-x: auto; scroll-snap-type: x mandatory with 6 <a> cards
Each card: numbered (01-06), with gradient background, accent color per card, title, category label, "View project →" link — all from the Claude Design's markup
data-anchor-progress attribute retained for deep linking
2. Replace Motion CSS in app/globals.css
Remove the entire .motion CSS block (~lines 763–1024): .motion, .motion-sticky, .motion-heading, .motion-link, .motion-smil-stage, .motion-study, .motion-project-grid, .motion-project-card, .motion-project-art, .motion-project-meta, .motion-orb, all [data-motion-renderer], [data-motion-layout='static'], and mobile/reduced-motion overrides.

Add new .motion CSS matching the Claude Design:

Dark section background with radial gradient (rgba(60,108,255,.14))
Typography using existing design tokens (--font-mono for eyebrow, --font-display for h2)
Horizontal scroll rail: hidden scrollbar (.swrail), scroll-snap-type: x mandatory, padding: 6px 6vw
Card styling: flex: 0 0 clamp(19rem,25vw,22rem), rounded 14px, gradient backgrounds per card, large ghost numbers (9rem), border glow on hover (translateY(-8px) + border-color change), scroll-snap-align: start
Responsive: cards shrink on mobile, text scales via clamp()
prefers-reduced-motion: reduce disables hover transitions
